## 2. 주요 개선 내역 및 버그 해결 기록

### 1. 일/주/월 다중 타임프레임 차트 구현
* **내용**: 상세 조회 API(`/api/stocks/detail`)가 단일 호출 시 `Promise.all`을 이용해 일봉, 주봉, 월봉, 분봉 데이터를 동시에 병렬 수집하여 리턴하도록 설계했습니다.
* **효과**: 탭 전환 시 새로운 네트워크 요청 딜레이 없이 즉각적이고 매끄러운(Lag-free) 차트 스위칭이 가능하여 훌륭한 사용자 경험을 제공합니다.
* **디자인 고도화**: 차트 툴바의 필 선택기(`pillSelector`) 영역의 가로폭을 `220px`로 늘리고 일, 주, 월, 분 4가지로 탭 버튼을 깔끔하게 재배치했습니다. 이동평균선의 접미사도 탭에 맞게 `5일`, `5주`, `5월`, `5분` 형태로 실시간 변환됩니다.
