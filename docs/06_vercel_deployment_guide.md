# Vercel(버셀) 웹사이트 배포 가이드 (초보자용)

이 문서는 IT 지식이 없는 분들도 단계별로 쉽게 따라 하며 프로젝트를 Vercel(버셀)에 배포할 수 있도록 안내하는 가이드입니다. 

---

## 1. 배포 전 준비사항

배포를 위해 다음 두 가지 계정이 준비되어 있어야 합니다.

1. **GitHub(깃허브) 계정**: 프로젝트 소스 코드가 저장되는 곳입니다.
2. **Vercel(버셀) 계정**: 소스 코드를 홈페이지 형태로 변환하여 인터넷에 띄워주는 서비스입니다.

> [!TIP]
> Vercel에 회원가입을 할 때는 **GitHub 계정으로 시작하기(Continue with GitHub)**를 선택하시면 나중에 연결하기가 훨씬 편리합니다.

---

## 2. 5분 만에 끝내는 배포 단계

### 1단계: Vercel 로그인
1. [Vercel 홈페이지(vercel.com)](https://vercel.com)에 접속합니다.
2. 화면 우측 상단의 **[Log In]**을 누르고, **[Continue with GitHub]**을 클릭하여 로그인합니다.

### 2단계: 내 저장소 불러오기
1. 로그인 완료 후 첫 대시보드 화면에서 우측 상단의 파란색 **[Add New...]** 버튼을 클릭하고 **[Project]**를 선택합니다.
2. 왼쪽의 **Import Git Repository** 목록에서 배포하고자 하는 프로젝트 저장소(`ai-pocketstock`)를 찾습니다.
3. 저장소 이름 오른쪽에 있는 **[Import]** 버튼을 클릭합니다.

> [!NOTE]
> 저장소 목록이 보이지 않는 경우, 화면 중앙의 **[Adjust GitHub App Permissions]** 링크를 클릭하여 Vercel이 깃허브 저장소를 볼 수 있도록 허용해 주세요.

### 3단계: 기본 설정 및 배포 실행
1. 프로젝트를 불러오면 상세 설정 화면이 나타납니다.
2. **이 프로젝트는 설정을 바꿀 필요가 없습니다.** Vercel이 웹사이트 유형(Next.js)을 자동으로 알아내어 설정해 줍니다.
3. 화면 가장 아래에 있는 파란색 **[Deploy]** 버튼을 클릭합니다.

### 4단계: 배포 완료 및 확인
1. 빌드 과정이 시작되며 1~2분 정도의 시간이 소요됩니다.
2. 화면에 종이꽃가루(Confetti) 효과와 함께 **"Congratulations!"** 문구가 나타나면 배포가 끝난 것입니다.
3. 화면에 표시되는 웹사이트 이미지(미리보기)를 클릭하거나 제공되는 인터넷 주소(예: `...vercel.app`)를 클릭하면 인터넷에서 실시간 주식 대시보드가 정상 작동하는 것을 확인할 수 있습니다.

---

## 3. 이것만 알아두세요!

* **자동으로 사이트가 업데이트됩니다**:
  앞으로 개발자가 코드를 수정하여 GitHub에 저장(Push)을 하면, Vercel이 이를 실시간으로 감지하여 **자동으로 새 버전으로 사이트를 다시 배포**해 줍니다. 여러분이 별도의 배포 작업을 매번 하실 필요가 없습니다.
